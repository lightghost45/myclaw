# TOOLS.md - 本机备忘

技能（skills）决定“怎么用工具”。这个文件记录“这台机器/你的习惯”的专属信息。

## 这里写什么

比如：

- 摄像头名称与位置
- SSH 主机别名
- TTS 声音偏好
- 房间/设备昵称
- 任何环境相关的细节

## 示例

```markdown
### 摄像头
- living-room → 客厅主区域，180° 广角
- front-door → 入户门，支持移动侦测

### SSH
- home-server → 192.168.1.100, user: admin

### TTS
- 默认声音：Nova（温暖，略带英式）
```

## 为什么单独放

skills 是通用的；这台机器的细节是私有的。
把它们分开，方便升级 skills，也避免泄露基础设施信息。
